# Rust Blockchain Explorer

A high-performance blockchain explorer built with Rust, compiled to WebAssembly for blazing-fast client-side performance.

## Project Overview

This blockchain explorer allows users to browse transactions, blocks, and accounts on various blockchains with exceptional performance and a pixel-art aesthetic UI that matches my personal brand.

## Technical Details

### Frontend
- Built with Rust + WebAssembly
- Uses Yew framework for component-based architecture
- Custom pixel art styling with responsive design

### Backend
- Rust-based API server using Actix-web
- Fast blockchain data indexing with custom parsers
- Multi-chain support (Ethereum, Solana, and more)

### Key Features
- Real-time transaction monitoring
- Account history with visualization
- Gas price analytics
- Smart contract interaction
